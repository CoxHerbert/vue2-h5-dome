# Codex 操作须知

在开始任何一次 Codex 作业之前，请务必完整阅读本文件，并在执行过程中遵守以下约定：

1. **首要任务**：再次确认当前需求是否涉及文档或指令文件的更新，若有请优先保证说明文档的准确性与一致性。
2. **技术栈约定**：项目基于 Vue 3 + Vite + Vant 4，依赖管理使用 `npm`。涉及依赖变更时请同步更新 `package.json` 与 `package-lock.json`。
3. **代码风格**：遵循 ESLint 与 Prettier 约束，不得随意禁用规则；新增脚本或配置需在 README 的“常用脚本”或相关章节补充说明。
4. **提交要求**：所有改动需包含充分说明（更新 README 或其它文档时保持中英文混排格式一致），并在本地通过必要的测试或说明未运行原因。
5. **安全与凭据**：切勿在仓库内写入真实敏感信息，涉及接口域名或密钥请使用环境变量或模拟数据。

阅读并理解上述内容后再开始本次改动，确保后续提交始终满足这些约定。
